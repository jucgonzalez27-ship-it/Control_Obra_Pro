# Control Obra Pro

Estado: DEFINIDO

Control Obra Pro es una herramienta para controlar el avance operativo de departamentos en obra de edificación.

No es un checklist aislado. El foco es responder rápidamente:

- Qué falta para terminar.
- Qué está bloqueando.
- Quién debe actuar.
- Qué puede liberarse.

## Alcance V1

Implementar solamente:

- Configuración de obra.
- Tipologías.
- Recintos.
- Partidas.
- Especialidades.
- Dependencias críticas.
- Avance.
- Dashboard.
- Centro de Comando.
- Reportes.
- Liberación.
- Historial.

## Fuera De Alcance V1

No implementar:

- IA.
- Chat.
- COP.
- Predicciones.
- Calidad avanzada.
- Compras.
- Bodega.
- Postventa.
- Multiobra.
- Aplicación móvil.

## Principio Operativo

El valor del producto está en priorizar acción diaria de obra, no en mostrar porcentajes por sí mismos.
