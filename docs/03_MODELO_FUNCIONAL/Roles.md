# Roles

Estado: DEFINIDO

## Administrador

Puede:

- Configurar.
- Actualizar estados.
- Verificar.
- Marcar No aplica.
- Liberar departamento.

## Supervisor

Puede usar:

- No iniciada.
- En proceso.
- Terminada.
- Observada.
- Bloqueada.

No puede:

- Verificada.
- No aplica.
- Liberar departamento.

## Solo Lectura

No puede modificar información.
