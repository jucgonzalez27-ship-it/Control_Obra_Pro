# Recinto: Dormitorio

Estado: DEFINIDO

## Regla

Dormitorio hereda la matriz de dependencias de Living - Comedor.

Todos los dormitorios consideran Radiador.

El closet normal dentro del dormitorio se modela como partida del recinto Dormitorio.

## Checklist V1

- Tabique.
- Empaste.
- Aparejo.
- Primera mano de pintura.
- Mano final de pintura.
- Piso flotante.
- Guardapolvo.
- Cornisa.
- Junquillo.
- Puerta.
- Cerradura.
- Closet.
- Componentes eléctricos.
- Radiador.

## Closet

Si un dormitorio no tiene closet, la partida debe quedar como No aplica o no cargarse en checklist según tipología.

PENDIENTE DE DEFINICIÓN: regla definitiva para cada tipología.
