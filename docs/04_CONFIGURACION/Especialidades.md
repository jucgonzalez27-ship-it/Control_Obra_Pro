# Especialidades

Estado: DEFINIDO

## Objetivo

Permitir responder qué especialidad debe actuar sobre una partida pendiente.

## Especialidades V1

- Pintura.
- Cerámicos.
- Piso Flotante.
- Guardapolvos.
- Cornisas y Junquillos.
- Puertas.
- Muebles.
- Cubiertas.
- Vidrios y Barandas.
- Electricidad.
- Sanitario.
- Climatización.
- Limpieza.
- Terminaciones Generales.

## Regla

Cada partida debe poder asociarse a una especialidad.
